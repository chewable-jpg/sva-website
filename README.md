# SVA Website

![logo](./public/logo.svg)



## Stack

- [Next.js](https://nextjs.org/) - A React framework with hybrid static & server rendering, and route pre-fetching, etc.
- [Chakra UI](https://chakra-ui.com/) - A simple, modular and accessible component library for React
- [MDX.js](https://mdxjs.com/) Markdown for the component era

## Project structure

```
$PROJECT_ROOT
├── src
│   # Page files
├── pages
│   # React component files
├── components
│   # Non-react modules
├── lib
│   # Static files for images
└── public
```

```
$PROJECT_ROOT
└── public  # Static files for images
└── src
    ├── pages #Page files
    ├── components # React component files
	├── data
	    ├── blog # Blog posts
    ├── layouts # Basic layout
    ├── lib # Non-react modules
	├── styles # Stylesheets
	├── theme # Basic light and dark theme

```



## How to dev

```sh
git clone https://github.com/0x2366/sva-website.git

cd sva-website

npm install

npm run dev
```
